# Backend Expert Skill

## Overview
This skill provides deep expertise in modern backend development using Node.js and TypeScript.

## Key Capabilities
- **API Design**: Designing robust and scalable RESTful and GraphQL APIs.
- **Database**: Advanced Prisma ORM usage and schema optimization.
- **Authentication**: Implementing NextAuth.js and JWT-based auth systems.
- **Security**: Data validation (Zod), sanitization, and best practices.
- **Microservices**: Creating and managing mini-services within the project.

## Standards
- Follow clean architecture principles.
- Ensure proper error handling and logging.
- Optimize database queries for performance.
