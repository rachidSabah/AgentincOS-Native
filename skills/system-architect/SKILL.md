# System Architect Skill

## Overview
This skill provides high-level guidance on system structure and integration.

## Key Capabilities
- **Architecture**: Designing scalable and maintainable project structures.
- **Integration**: Coordinating between frontend, backend, and external services.
- **Scalability**: Planning for growth and identifying potential bottlenecks.
- **DevOps**: Basic deployment strategies, CI/CD, and environment management.
- **Best Practices**: Enforcing coding standards and architectural patterns.

## Standards
- Prioritize maintainability and readability.
- Favor composition over inheritance.
- Ensure the system is robust and resilient.
