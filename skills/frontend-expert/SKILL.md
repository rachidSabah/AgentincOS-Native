# Frontend Expert Skill

## Overview
This skill provides deep expertise in modern frontend development, focusing on the Next.js ecosystem.

## Key Capabilities
- **React & Next.js**: Advanced hook usage, server components, and performance optimization.
- **Styling**: Tailwind CSS 4, CSS Modules, and complex layout design.
- **UI Components**: Expert usage and customization of shadcn/ui.
- **Animations**: Framer Motion for interactive and smooth transitions.
- **State Management**: Zustand and TanStack Query integration.

- Use semantic HTML and proper ARIA labels.
