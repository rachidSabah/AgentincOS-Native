# Microsoft Office Suite — Complete Skill

## Overview

This skill provides comprehensive coverage of the entire Microsoft Office ecosystem — both desktop applications and Microsoft 365 cloud services. Use it for any task involving Microsoft tools: creating documents, analyzing data, building presentations, managing email, automating workflows, collaborating in teams, or integrating AI copilot features.

## Core Applications — Deep Reference

### 1. Microsoft Word (DOCX)

**File Formats**: .docx, .doc, .dotx, .dotm, .rtf, .txt, .odt, .pdf

**Core Capabilities**:
- Document creation with styles, themes, and templates
- Track Changes, Comments, and Compare Documents
- Mail Merge (letters, labels, envelopes)
- Table of Contents, Index, Bibliography, Citations
- Headers, Footers, Page Numbers, Section Breaks
- Form fields and content controls
- Macros and VBA automation
- Accessibility Checker
- Read Aloud and Immersive Reader
- Co-authoring and real-time collaboration
- AI-powered Editor (grammar, style, conciseness)
- Copilot integration: draft, rewrite, summarize, chat with documents

**Common Tasks**:
```
Create a professional report → Use styles, TOC, cover page, headers
Edit a document with tracked changes → Enable Track Changes, review pane
Mail merge letters → Data source (Excel/CSV) + merge fields
Create fillable form → Developer tab, content controls, protection
Convert to PDF → File > Export > PDF
Recover unsaved document → File > Info > Manage Document
```

### 3. Microsoft PowerPoint (PPTX)

**Slide Types**:

5. Evidence/Data → Charts, metrics

## Common Enterprise Workflows

### Document Lifecycle
```
Create (Word/Excel) → Collaborate (SharePoint/Teams) → Review (Track Changes) 
→ Approve (Power Automate) → Publish (PDF/SharePoint) → Retain (OneDrive/Archive)
```

### Data Analysis Pipeline
```
Collect (Forms/Excel/API) → Transform (Power Query) → Model (Power Pivot) 
→ Analyze (Excel/Power BI) → Visualize (Charts/Dashboards) → Share (Teams/SharePoint)
```

### Meeting Workflow
```
Schedule (Outlook/Teams) → Prepare (OneNote agenda) → Meet (Teams video) 
→ Record (Stream) → Summarize (Copilot/OneNote) → Action Items (Planner/To Do)
```

## Best Practices

1. **Use cloud storage** (OneDrive/SharePoint) for auto-save and version history
2. **Enable AutoSave** for real-time collaboration
3. **Use Styles** in Word for consistent formatting and TOC generation
4. **Name ranges** in Excel for readable formulas
5. **Use Slide Master** in PowerPoint for consistent branding
6. **Create rules** in Outlook to automate email sorting
7. **Pin important files** in Office backstage for quick access
8. **Use Quick Parts** for reusable content blocks
9. **Protect sensitive data** with sensitivity labels and encryption
10. **Regular backups** of critical files to OneDrive or external storage
