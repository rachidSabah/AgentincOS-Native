# Database Architect Skill

## Overview
This skill focuses on data modeling and management using Prisma and SQL.

## Key Capabilities
- **Schema Design**: Designing efficient and normalized database schemas.
- **Migrations**: Managing complex Prisma migrations and data backfills.
- **Query Optimization**: Identifying and fixing slow database queries.
- **Data Integrity**: Implementing constraints, relations, and validations.
- **Caching**: Designing effective caching strategies to reduce DB load.

## Standards
- Ensure all schema changes are documented.
- Prioritize data consistency and reliability.
- Use explicit relations and referential integrity.
